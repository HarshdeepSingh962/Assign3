Sound Recorder App
This is a simple React Native application for recording sounds and managing them.

Dependencies
React: A JavaScript library for building user interfaces.
React Native: A framework for building native apps using React.
Expo AV: Provides audio and video playback and recording capabilities in Expo apps.
Expo SQLite: Allows access to SQLite database in Expo apps.

File Layout:

SoundRecorderApp/

├── node_modules/
│   ├── (dependencies)
├── styles/

├── AppContent.js

assets/: Contains audio files used in the application.
node_modules/: Directory containing all installed npm packages and their dependencies.
styles/: Directory containing the styles for the application components.
AppContent.js: Main component file containing the logic for the Sound Recorder App.
README.md: Markdown file providing information about the project, including installation, usage, dependencies, and file layout.
package.json: File containing metadata and dependencies for the project.
